# Campaign Pipeline Workflow

## Overview

The Campaign Pipeline is a comprehensive workflow system for managing creator outreach, negotiation, contracting, and campaign execution. It provides end-to-end automation with AI-powered sentiment analysis and negotiation assistance.

This workflow is designed based on industry best practices for influencer/creator marketing, covering all phases from discovery to post-campaign analysis.

**5.3 Pricing Benchmarks (Industry Average)**
| Tier | Followers | Instagram Post | TikTok | YouTube |
|------|-----------|----------------|--------|---------|
| Nano | 1K-10K | $50-$250 | $50-$200 | $200-$500 |
| Micro | 10K-50K | $250-$1,000 | $200-$800 | $500-$2,000 |
| Mid-Tier | 50K-500K | $1,000-$5,000 | $800-$3,000 | $2,000-$10,000 |
| Macro | 500K-1M | $5,000-$15,000 | $3,000-$8,000 | $10,000-$30,000 |
| Mega | 1M+ | $15,000-$100K+ | $8,000-$50K+ | $30,000-$100K+ |

**5.4 Counter-Offer Strategies**
- Value justification
- Bundle deliverables
- Offer performance bonuses
- Extend partnership duration
- Provide exclusive access
- Include product value
- Negotiate usage rights separately

**5.5 🤖 AI Auto-Negotiation Feature**

The platform includes an AI-powered auto-negotiation system that can handle price negotiations automatically based on your configured parameters.

**Auto-Negotiation Settings:**
- **Strategy Selection:**
  - **Aggressive**: Small increments (10%), stay closer to initial offer
  - **Balanced**: Meet in the middle approach (30% movement toward ask)
  - **Generous**: Move toward creator ask faster (50% movement)

- **Budget Parameters:**
  - Target Budget: Ideal price to aim for
  - Minimum Budget: Floor for negotiations
  - Maximum Budget: Ceiling for negotiations
  - Auto-Decline Above: Automatically decline if ask exceeds threshold

- **Automation Rules:**
  - Auto-Accept Threshold: Accept if creator ask is within X% of target
  - Increment Per Round: How much to increase offer each round
  - Maximum Rounds: Escalate for manual review after N rounds

- **Smart Adjustments:**
  - Consider engagement rate for pricing adjustments
  - Consider follower count for tier-based offers

**How It Works:**
1. Enable auto-negotiation in campaign settings
2. Configure strategy, budget ranges, and thresholds
3. When a creator responds with a counter-offer:
   - AI analyzes the response and extracts pricing
   - Calculates optimal counter-offer based on market rates
   - Applies configured strategy and constraints
   - Sends personalized negotiation email automatically
4. Negotiations continue until:
   - Agreement is reached (within acceptable range)
   - Maximum rounds exceeded (escalates to manual review)
   - Creator accepts or declines

**AI-Powered Features:**
- Market rate calculation based on creator metrics
- Intelligent counter-offer generation
- Email template personalization
- Sentiment analysis of responses
- Automatic escalation for edge cases

## 📊 Pipeline Stages

```
1. OUTREACH → 2. AWAITING_RESPONSE → 3. RESPONSE_RECEIVED → 4. NEGOTIATION →
5. AGREEMENT_REACHED → 6. CONTRACT_SENT → 7. CONTRACT_SIGNED → 8. IN_PROGRESS → 9. COMPLETED
                                                              ↓
                                                          DECLINED / NO_RESPONSE
```

### Stage Descriptions

| Stage | Description | Actions Available |
|-------|-------------|-------------------|
| **outreach** | Initial state, ready to send outreach email | Send outreach email |
| **awaiting_response** | Email sent, waiting for creator reply | Send follow-up (after 3 days) |
| **response_received** | Creator has responded | Analyze sentiment, start negotiation |
| **negotiation** | Price/terms negotiation in progress | Send counter offer, finalize deal |
| **agreement_reached** | Terms agreed upon | Generate contract, send for signing |
| **contract_sent** | Contract sent to creator | Send reminders, track viewing |
| **contract_signed** | Creator has signed the contract | Start campaign, send brief |
| **in_progress** | Campaign is live | Track deliverables, provide feedback |
| **completed** | Campaign finished successfully | Add review, send thank you |
| **declined** | Creator declined the offer | Archive, note for future |
| **no_response** | No response after 3 follow-ups | Archive |

## 📈 Analytics & Metrics

The pipeline tracks:
- **Response rate**: % of creators who responded
- **Average response time**: Time from outreach to response
- **Negotiation success rate**: % of negotiations that succeeded
- **Average negotiation duration**: Time to reach agreement
- **Total budget spent**: Sum of all agreed prices
- **Average sentiment score**: Overall creator interest level
- **Completion rate**: % of creators who completed campaigns
- **Creator review scores**: Quality ratings from past collaborations
